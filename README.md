# SmplOSV2
Reworked Smplos WIP

What is done:

  --> GDT
  
  --> Interrupts (ISR's and IRQ's)
  
  --> Memory Allocation (Kernel is lower half, memory at 0xFFC00000 is reserved for page tables), memory map obtained from grub
  
  --> VFS
  
  --> Very basic LibC
  
  --> CMOS & PIT clocks
  
  --> Basic scheduling
